# 术语介绍

如果您刚开始对接喜马拉雅的Marketing API，那么本节将帮助你快速了解喜马拉雅Marketing API相关术语的介绍和接口的说明

|  术语名   | 中文名 | 描述 |
|  ----  | ---- | ---- | 
|  Marketing API  | 开发者平台 | 要获取Marketing API的接入权限，您首先需要注册成为开发者 | 
|  ad_account_id  | 广告账户ID | 对应广告投放平台的广告账户ID | 
|  refresh_token  | 刷新token | 开发者会被分配一个refresh_token，用于登录刷新access_token |
