# 错误码

**返回示例 : 成功**

```json
{
    "status": 0,
    "type": "SUCCESS",
    "msg": "success",
    "data": ""
}

```

**返回示例 : 失败**

```json

{
    "status": 10010003,
    "type": "ERROR",
    "msg": "权限校验不通过",
    "data": null
}

```

|  错误码(code)   | 描述  | 
|  ----  | ---- | 
| 10003  | 权限校验不通过 | 
| 10011  | 登录失败 |
| 50000  | 未知错误 |
| 50001  | 入参校验错误 |
