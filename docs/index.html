    Ximalaya ADS

    Marketing API

    通过本篇您可以了解喜马拉雅Marketing API使用的基础知识，使用前您需要注册并申请成为开发者，接下来创建应用以获得refresh_token，当这两步都完成后，就可以向广告主申请授权，当广告主同意授权后您会获得对应的授权码，接下来使用授权码获取相应access-token就可以使用API管理账号进行Marketing API的调用了。

    Ximalaya Ad-Exchange

    如果您是程序化的demand/buyer侧，需要获取喜马拉雅的广告请求流量，请参见https://github.com/xmlyads/adx

    OpenRTB API

    如果您是程序化的supply侧，需要获取喜马拉雅的广告预算，请查看https://xmlyads.github.io/openrtb/statements/
