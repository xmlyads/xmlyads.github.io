    实时竞价接口

    实时竞价接口

    1: 基本协议信息

    请求header:

    SSP采用HTTP通信协议与喜马联盟广告平台交互，广告请求(Request)与响应(Response)使用protobuf(Google Protobuf V3)进行序列化

    基本协议 说明
    HTTP METHOD POST
    Content_Type application/x-protobuf
    广告请求URL 测试和正式域名， 请联系喜马运营

    2: 广告请求（BidRequest）

    2.1: 协议基本信息

    字段名 类型 是否必须 说明
    api_version string Required 协议版本（目前为"v1.0"）
    id string Required 本次请求的唯一id
    ts int64 Required 请求unix时间戳，单位毫秒
    imp Imp Required 广告位数组
    device Device Required 设备信息
    user User Required 用户信息
    app App Required APP信息
    extra Ext Optional 附加信息

    2.2: Imp（广告位信息）

    字段名 类型 是否必须 说明
    id string Required 标识唯一一次曝光机会
    ad_type int32 Required 广告类型（1:开屏、2：信息流、3：激励视频、4：贴片、5：banner）
    tag_id string Required 广告位id
    bidfloor float Optional 底价，元/千次曝光
    template_id string Required 喜马联盟广告模板模板id

    2.2.3: Device（设备信息）

    字段名 类型 是否必须 说明
    ua string Required 设备UA信息
    ip string Required 设备出口IP地址
    os int32 Required 设备类型（0：Android；1：iOS;2：IPAD;3：androidPad）
    carrier int32 Required 运营商（0：移动；1：联通;2：电信；3：其它）
    net int32 Required 网络连接类型（0：2G；1：3G；2：4G；3：5G；4：wifi；6：其他）
    make string Required 手机品牌，如：iPhone，huaWei
    model string Required 手机型号，如：iPhone15，RMO-AN00
    h string Required 设备屏幕纵向分辨率
    w string Required 设备屏幕横向分辨率
    geo Geo Optional 地理位置信息
    idfa string Optional IDFA原始值
    idfa_md5 string Optional IDFA MD5值
    imei string Optional IMEI原始值
    imei_md5 string Optional IMEI MD5值
    oaid string Optional OAID原始值
    oaid_md5 string Optional OAID MD5值
    android_id string Optional Android Id原始值
    android_id_md5 string Optional Android Id MD5值
    caid string Optional CAID原始值
    caid_md5 string Optional CAID MD5值
    caid_version string Optional CAID版本
    mac string Optional MAC地址原始值
    mac_id string Optional MAC地址 MD5值
    2.3: GEO（地理位置）
    字段名 类型 是否必须 说明
    type int Optional 经纬度有值时必传，坐标系类型 0:GCJ-02; 1:WGS-84; 2:bd09ll
    lat string Optional 维度
    lon string Optional 经度

    2.4: App（APP信息）

    字段名 类型 是否必须 说明
    bundle string Required 包名
    app_name string Required APP名称
    app_version string Optional APP自身版本号

    3：广告响应（BidResponse）

    3.1: 响应结果

    字段名 类型 是否必须 说明
    id string Required 协议版本（目前为"v1.0"）
    code int32 Required 广告响应状态码
    bid_id string Required 喜马联盟侧生成的竞价ID
    seat_bid seatBid Required 喜马联盟侧生成的竞价ID

    3.2: SeatBid（竞价的具体内容）

    字段名 类型 是否必须 说明
    bid Bid Required 与BidRequest.imp对应，每个imp最返回一个bid
    Bid（Bid信息）
    字段名 类型 是否必须 说明
    id string Required 喜马联盟侧生成
    imp_id string Required 曝光ID与BidRequest.impId一致
    tag_id string Required 广告位ID
    ad_id string Required 广告创意Id
    price float Required 价格、单位元
    adm Adm Required 创意物料信息
    impression_urls string Required 广告曝光的时候触发，竞价模式需要支持替换价格(__SETTLE_PRICE__宏替换)
    click_urls string Required 广告点击是触发的时候触发
    win_notice_url string Optional 竞价成功通知url，需支持获胜/未获胜宏替换
    Adm
    字段名 类型 是否必须 说明
    template_id string Required 模板ID
    title string Optional 标题
    image Image Optional 图片信息
    video Video Optional 视频信息
    dpLink string Optional dp唤起连接，优先dp
    link string Optional 网址落地页
    desc string Optional 描述
    Image
    字段名 类型 是否必须 说明
    url string Required 图片url
    width string Optional 宽度
    height Image Optional 高度
    mime Video Optional 图片类型：jpg、jpeg、png、gif
    Video
    字段名 类型 是否必须 说明
    url string Required 图片url
    width string Optional 宽度
    height Image Optional 高度
    mime Video Optional 视频类型：mp4
    duration Video Optional 视频时长
    size Video Optional 视频大小，单位Byte

    4: 监测

    4.1: 曝光

    impression_urls 为曝光监测地址，⼴告⼀经曝光，就会⽴即上报该数组的链接，上报会从用户手机上发起，上报⽅式为http get请求。如需要相关宏替换的操作，⻅1.4.4支持的宏替换。

    4.2: 点击

    click_urls 为点击监测地址，⼴告⼀经曝光，就会⽴即上报该数组的链接，上报会从用户手机上发起，上报⽅式为http get请求。如需要相关宏替换的操作，⻅1.4.4支持的宏替换。

    4.3: 竞价成功/失败

    win_notice_url 为竞价成功通知地址，需要对结算价格字段(${AUCTION_PRICE})进行宏替换

    4.4: 支持的宏替换

    宏含义 宏 规范与取值
    结算价格 __SETTLE_PRICE__ 单位：分/cpm，采用AES加密，模式使用：ECB，填充：pkcs5padding，数据块：128位；输出：hex；字符串：utf8；支持宏替换url：impression_urls、click_urls
    时间戳 __TS__ 精确到毫秒级，例如：1589168735000
    曝光区域 __IMP_AREA_X1Y1X2Y2__ 左上角(X1,Y1), 右下角(X2,Y2)
    点击区域 __upX__, __upY__, __downX__, __downY__ 点击坐标单位为像素、数值为素材图片左上角为起点的X和Y轴 __DOWN_X__：用户手指按下时的横坐标 __DOWN_Y__：用户手指按下时的纵坐标 __UP_X__：用户手指离开设备屏幕时的横坐标（目前上报的值=按下的横坐标）__UP_Y__：用户手指离开设备屏幕时的纵坐标（目前上报的值=按下的纵坐标）
    结算价格 ${AUCTION_PRICE} 单位：分/cpm，采用AES加密，模式使用：ECB，填充：pkcs5padding，数据块：128位；输出：hex；字符串：utf8
    竞价失败原因 ${LOSS_MESSAGE} 竞价失败的原因
