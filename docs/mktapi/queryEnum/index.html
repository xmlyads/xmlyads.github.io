    枚举信息列表

    请求地址 ： https://catyph.ximalaya.com/ads-catyph/v1/system-enum/get

    请求方式 ： GET

    请求头参数 ：

    字段名 类型 是否必须 描述
    access-token string 是 鉴权用access-token

    请求参数 ：

    传参形式 ： Url Param

    字段名 类型 是否必须 描述
    version string 是 版本号

    返回字段释义：

    字段名 类型 描述
    status int 状态码 0：正常 ， 其他：异常
    type string 返回类型
    msg string 返回信息
    data object 返回数据
    - enum array 枚举类数组
    -- name string 枚举值名
    -- desc string 枚举值含义描述
